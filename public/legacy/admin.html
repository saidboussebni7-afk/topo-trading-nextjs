🔥 إعداد Firebase
خطوات إعداد Firebase:

1. اذهب إلى console.firebase.google.com
2. أنشئ مشروع جديد
3. اذهب إلى Project Settings → General
4. في قسم "Your apps" أضف Web App
5. انسخ بيانات firebaseConfig
6. اذهب إلى Firestore Database وأنشئه في وضع Test Mode
7. ضع البيانات في الحقول أدناه
